KEY DEPENDENCIES EXPLAINED
================================================================================

GSAP (GreenSock Animation Platform):
  - Industry-standard JavaScript animation library
  - Used for: timeline animations, scroll triggers, complex sequences
  - Components: gsapSetup.js handles initialization
  - Key Methods: gsap.timeline(), ScrollTrigger

Framer Motion:
  - React-native animation library
  - Used for: component-level animations, staggered effects
  - Key Features: Keyframes, layout animations, variants

Three.js & React Three Fiber:
  - 3D graphics rendering engine
  - Used for: journey visualization, 3D scenes, terrain
  - react-three/fiber provides React bindings
  - drei provides helpful components (OrbitControls, etc.)

Lenis:
  - High-performance smooth scrolling library
  - Provides: event-based scroll updates, inertia
  - Integration: useSmoothScroll hook

Rive:
  - Vector animation and graphics editor
  - Used for: interactive animated graphics
  - Assets located in: public/rive/

PERFORMANCE CONSIDERATIONS
================================================================================

1. Code Splitting:
   - JourneyCanvas is lazy-loaded with React.lazy()
   - Reduces initial bundle size
   - Loaded only when needed

2. Asset Optimization:
   - SVG assets in public/svg/
   - Rive animations pre-optimized
   - Images placeholder component

3. Animation Performance:
   - GSAP uses requestAnimationFrame
   - GPU-accelerated transforms (transform, opacity)
   - Avoid animating expensive properties (width, height)

4. Scroll Performance:
   - Lenis handles smooth scrolling efficiently
   - ScrollStages manages scroll stage tracking
   - Waypoint component uses Intersection Observer API

5. 3D Performance:
   - Three.js with proper LOD (Level of Detail)
   - Particle systems optimized
   - Terrain generated procedurally to reduce assets

TROUBLESHOOTING
================================================================================

Port Already in Use:
  - Change port in vite.config.js
  - Or kill process using port 5173

Animations Not Triggering:
  - Check GSAP ScrollTrigger refresh in gsapSetup.js
  - Verify ScrollStages initialization
  - Check MotionModeContext setting

3D Scene Not Rendering:
  - Verify WebGL support in browser
  - Check browser console for errors
  - Ensure Three.js dependencies installed

Build Fails:
  - Remove node_modules and package-lock.json
  - Run npm install again
  - Check Node.js version compatibility
